## 📦 What is aibot-python-sdk?

The **aibot-python-sdk** is a simple tool that helps you connect with WeCom AI Chatbots using Python. It lets you send and receive messages through your work chat system. This SDK works as a bridge between you and the AI bot, making chatbot use smooth and straightforward.

This software is designed for Windows computers. You do not need to know programming to get it running. This guide will show you how to download, install, and start using the SDK step by step.

---

## 💻 System Requirements

Before you start, make sure your computer meets these needs:

- Operating System: Windows 10 or later
- Python: Version 3.6 or newer (you will find how to install this below if needed)
- Free disk space: At least 100 MB
- Internet connection: Required to download and use the SDK
- Basic knowledge of Windows file system recommended but not necessary

## 🚀 Getting Started

Follow these steps to run the **aibot-python-sdk** on your Windows computer.

## 📥 Step 1: Download and Unpack

1. Open the download page by clicking the button above.
2. On GitHub, find the “Code” button near the top right.
3. Click “Download ZIP” to save the project files to your computer.
4. Once the download finishes, locate the `.zip` file (usually in your "Downloads" folder).
5. Right-click the `.zip` file and select “Extract All.”
6. Choose a folder you can easily find, like your Desktop.
7. Click “Extract” and wait for the files to appear.

---

## 🐍 Step 2: Install Python

Python is the software language behind the SDK. You need it to run the program.

1. Check if Python is already installed:
   - Click the Start menu
   - Type `cmd` and open the “Command Prompt” app
   - In the window, type `python --version` and hit Enter
2. If it shows a version number like `Python 3.8.5`, you can skip to Step 3.
3. If not installed, here is how:
   - Visit the official Python website: https://www.python.org/downloads/windows/
   - Click on the latest Python 3 release for Windows.
   - Download the "Windows installer" (choose 64-bit if unsure).
4. Run the installer and **make sure to check "Add Python to PATH"** before clicking “Install Now.”
5. Wait for the installation to complete.

---

## ⚙️ Step 3: Install Required Libraries

The SDK uses some extra tools called libraries. Installing these lets the program work properly.

1. Open Command Prompt again (see Step 2 for help).
2. Type the following command exactly and hit Enter:

   ```
   pip install requests websocket-client
   ```

3. Wait while it installs. This might take a minute.
4. When done, you are ready for the next step.

---

## ▶️ Step 4: Run the SDK

1. Open the folder where you extracted the files (from Step 1).
2. Find a file named something like `example.py` or `run.py`. This is usually the main file to launch the program.
3. Hold Shift, right-click within the folder, and choose “Open PowerShell window here” or “Open command window here.”
4. In the new window, type:

   ```
   python example.py
   ```

5. Press Enter to run.
6. The SDK will start, and you will see messages showing it is connecting to the WeCom AI bot.

---

## 🔍 How to Use the SDK

The aibot-python-sdk sends text to your AI chatbot and shows responses on your screen. You can type a message and get a reply.

- Type your question or message in the command window when prompted.
- Press Enter.
- The AI bot will respond with an answer.
- Continue chatting this way.

## ⚙️ Configuration

For the SDK to connect correctly, you may need to set your WeCom credentials:

- Open the `config.ini` or similar file in a text editor (Notepad works fine).
- Enter your company ID, secret key, or API token.
- Save the file before running the program.

If you do not have these credentials, contact your workplace IT or WeCom administrator.

---

## 🛠 Troubleshooting

If you encounter issues, try these tips:

- Make sure Python is installed and added to your PATH.
- Confirm all required libraries are installed using `pip install`.
- Check your internet connection.
- Restart the app if the connection to the AI bot fails.
- Verify your WeCom credentials in the config file.
